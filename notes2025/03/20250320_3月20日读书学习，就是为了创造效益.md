# 3月20日 读书学习，就是为了创造效益

天气：晴，5-19摄氏度，空气质量56，体感舒适

每日日常：多邻国✅扇贝单词✅力量训练+拉伸✅

继续精读“打开心智”，以我为主，为我所用。

触动语句：

“过于重视读很多本书，过于追求掌握每一本书，却很少有人真的思考：在阅读之后，我的看法是什么？我从中学到了什么东西”

感悟：

以前南北朝的时候，有个叫陆澄的人，号称两脚书橱，当时限于文字留存的书刊不多，他做到了无书不读；但后人，也就只了解到他的这个特征而已，其他什么了不起的事情，给民众留下什么贡献，那是一概没有。实际上，他有负责编撰一些书籍，也是拖拖拉拉，多年不成，死读书有用吗？没有。

打开心智里提到的症结之一，就是看了好多书，但就是为了读而读，形而上的东西多了，东一榔头西一锤。以前我也这样，出发点就是为了炫耀，哪本流行读哪本，把里面的一些金句摘录出来，动不动就甩几句出来，感觉老好；殊不知，在众位行家眼里，实在是小家子气了，路走偏了；从实际收益的角度来讲，也没有化为赚钱/提升能力的真正驱动，所谓知道了一些道理，仍然没有过好日子，就是这个意思。

说到底，读书就是为了有用，读一本书做到：学一个想法，并付诸实施成功，或增加思想力量，就已经很好；闲书乱读也不是不可以，但流于浅薄，改变生活的角度来看没用哈；要知道并不是所有内容，都值得花力气，去啃下来，反刍出啥道道来的。笔者最近和小伙伴在学习 python 编程知识，遇到不少坎，后来悟出来的方法就是多看学习视频和帖子，扣细节 / 再复原，也能够把模型跑起来，形成正反馈。
