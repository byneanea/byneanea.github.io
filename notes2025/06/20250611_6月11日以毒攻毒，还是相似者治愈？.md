今天学到一个单词，homeopathy，意思是顺势疗法。英语中，最复杂的部分是什么？是记名词，要了命的多，记不胜记。牛津词典有据可查的名词数量，轻松突破30万大关。

我喜欢的一种方法，会比较好玩些，挖掘词汇背后的故事，如果有的话；顺势疗法，就是一个有趣的例子。开始的时候，是德国的医生兼化学家，Samuel Hahnemann，1796年提出一种理论。

他在翻译一本讲解奎宁治疗疟疾的医学书时，突发奇想：“健康人吃了奎宁也会发冷发热，和疟疾症状一样？那是不是‘以毒攻毒’也能治病？”

于是，他开始“人体试毒”实验，试图用引起类似症状的药物，来治愈那些症状。他自己试着吃了奎宁（治疗疟疾的药），结果身体也出现了类似疟疾的发冷发热；就觉得：“健康人吃某物出现某症状 → 那东西就能治病人身上的这种症状。”

由此，他提出两大顺势疗法定律：

1. “相似法则”

   （like cures like），翻译成白话就是“能引发某种症状的东西，也能治愈这种症状。”
2. “最小剂量”

   越稀释越有力，“最小剂量”不是“吃少点药”，而是“喝几乎完全不含药的水”；相信药物的灵魂残留比药物本身还重要。

看过顺势疗法和现代医学的比对，咱们就能够知道，homeopathy更像一种信仰治疗，心理安慰，可以在现代医学触及不到的方面，给到信众一些无害的补充。

|  |  |
| --- | --- |
| 顺势疗法 | 现代医学 |
| “ 像感冒的东西能治感冒 ” | “ 杀病毒 + 缓解症状 ” |
| “ 药稀释到只剩水的记忆 ” | “ 药得精确、对路、对量 ” |
| “ 唤醒你体内的气场疗愈 ” | “ 直接出击，给你该补的补、该抑的抑 ” |
| “ 信念和自然力量 ” | “ 机制和数据支持 ” |
| “ 几乎无副作用，但可能也没效果 ” | “ 有效果，但要看说明书 ” |

为啥有人认为有用？因为：

- 实际上，多数疾病本来就能自愈

-  吃药后有“我在治疗”的心理安慰

-  顺势疗法不含活性成分=几乎无副作用

-  仪式感满满，像魔法一样酷炫

看完一个玄学理论的发展比对，回到单词本身：

Home是家，pathy源自  *pathos*  ，意思是“感觉、病、治疗”，o是连接的路人甲。

🧪 Homeopathy ： “在home喝水，path找能量，疗法靠‘拍一拍’！”

奇奇怪怪的姿势，又学废了一次，再也不怕忘记啦！
