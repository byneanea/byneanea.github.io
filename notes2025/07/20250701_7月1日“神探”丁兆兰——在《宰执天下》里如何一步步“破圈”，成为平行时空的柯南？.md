# 7月1日 “神探”丁兆兰——在《宰执天下》里如何一步步“破圈”，成为平行时空的柯南？

天气：青岛，多云，24-28摄氏度，空气质量20

每日日常：多邻国✅扇贝单词✅

更文269天/1088天

这不是普通的捕快传记，这是一篇把《七侠五义》+《名侦探柯南》+宋穿工业流，硬核搅拌在一起的，高能混剪话本。

丁兆兰是穿越小说“宰执天下”里的捕快，后世名侦探柯南一般的人物，在作者“酷死啦”的笔下，在下窃以为，是仅逊于男主角的存在；哪怕出场的时候，已经是快要完本的阶段，依然难掩王霸之气，秒破灭门案，三日破都堂枪击案，环环相扣，让喜爱公案画本，和现代侦探小说的书迷，开心不已。

1. 逻辑推理：把行人司的三辆作案马车，玩成“三套车”梗

📖 书中原文

“三辆车从头贯穿到尾……只有这辆马车是不要车夫，自付押金，也就是这一辆。”——《梳理，卷一四六》

🕵🏻‍♂️ 柯南平行宇宙

- 柯南爱画时间线，丁兆兰爱画“车辆流向图”。
- 柯南靠鞋底加速踢足球，丁兆兰靠缜密的大脑过滤车辆，
  两人都认准：排掉所有不合理，剩下的就是真相
  。

👀 看点

- 读者边翻页边想啊，“原来套牌车是这么查出来的！“；从此，目光扫到，但凡出租车牌子歪一点，都会联想起那三辆车。

2. 群众关系：把吃瓜百姓，升级成“朝阳区群众”

📖 书中原文

“这一位虽然只是个做小买卖的店主，但最爱的就是去茶馆听公案小说……过去兆兰手上有几桩案子，得了他的帮忙。”——《梳理，卷一四六》

🕵🏻‍♂️ 柯南平行宇宙

- 柯南有少年侦探团，丁兆兰有“茶客侦探团”。
- 让小贩、跑堂，主动化为“包打听”，比黑科技眼镜，更符合宋穿国情。

👀 看点

- 你不是龙套，你是“关键 NPC”；
- 有了破案的参与感，公案书迷嗑得嘎嘣脆，嗨翻天。

3.长官 buff：宰相韩冈专门“翻沙漏”给他

📖 书中原文

“这沙漏流完，大概是半小时多一点。时间都给你。”——《梳理，卷一四六》

🕵🏻‍♂️ 柯南平行宇宙

- 相当于警方把案发现场，完全交给“江户川柯南君”。
- 权力+时间
  ＝官方盖章的“想怎么断案就怎么断”。

👀 看点

宰辅圈，对他是“官方背书”；而粉丝圈，称他是“民间保护神”；

- 经此一案，“底层捕快”一跃成为
  警政改革的样板
  。

4.人性光辉：先让百姓退后，再跟贼人对刀

📖 书中原文

“俺是捕拿人犯！管你童子、和尚，犯了法，爷爷都捉！……俺只知道抓贼，不伤无辜。”——《梳理，卷一五一》

🕵🏻‍♂️ 柯南平行宇宙

- 柯南每次喊“大家快后退！” 然后踢球炸灯；
- 丁兆兰每次都是先清场，然后铁尺对唐刀，针尖对麦芒。
  动作不同，逻辑一致：群众安全第一。

👀 看点

- 捕快里罕见的“战斗佛”——杀气四溢但不滥伤无辜；
- 街坊心目中，他是：“活活一尊行走的平安佛”。

5.技术宅：指纹、枪管、车牌全是数据库思维

📖 书中原文

“看到锥形弹就知线膛；对比出厂编号，只有这一辆车嫌疑最大。”——（梳理，综合卷一四六）

🕵🏻‍♂️ 柯南平行宇宙

- 柯南随身带科技道具，丁兆兰注重“工具链”——

* 指纹对比、牙齿铸模；
* 车行出厂档案；
* 线膛枪弹道曲线。

👀 看点

- 在满地勾栏瓦肆的宋朝，硬生生掰出“近代法医现场”；让读者感叹：
  古代也能玩 CSI
  。

6.读者爽点：你不仅在看案子，你在参与案子

- 与大boss韩冈的“对决”：为生民，为求实，不惜赌上自己，依据逻辑链条，专业精神，和主角对质案情；跌宕起伏，精彩纷呈，不容错失的剧情。
- 公案彩蛋

  ：七侠五义人物名乱入（神探本人是双侠之一的名头，编排致死的四鼠，牵手共赢的欧阳春，艾虎，神枪手白玉堂） → 看得旧派武侠迷，大呼过瘾；
- 穿越科技

  ：线膛枪＆机器刻制的，马车套牌 → 工业党拍手称快。

🎯 一句话总结

丁兆兰 = 宋朝街头版柯南 + 展昭的江湖气 + 法医、刑警、情报官，多重职业buff集于一身。

他让老百姓真切感到——“原来破案也能这么自然而然！原来我也能出一分力！”

如果你喜欢 智商硬核、烟火味浓、又自带一点老武侠彩蛋 的侦探角色，丁兆兰绝对值得关注，喜欢宋穿题材的话，《宰执天下》（名字有点中二，但不妨碍是部好书）值得多翻几页。

下次再有人问： “宋朝真有神探吗？” 别犹豫，甩给他这句话👇——

“有！叫丁兆兰，连宰相都给他翻过沙漏，听案情分析。”

马伯庸老师曾经力荐本书，称其为“新宋”后最好看的宋穿小说，万维钢老师也在课程中提及本书的桥段， 认为：穿越小说除了穿越那一刻是神迹，接下来的演化都是真实的，是思想实验；而科幻小说是对物理定律的侮辱。当工作生活乏味疲劳的时候，翻上几页穿越剧本，转换一下场景，代入一下主角思维，也是非常好的换脑子方式。
